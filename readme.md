<h1 align="center">Jkeep by Laravel</h1><hr>
<p>해당 어플리케이션은 라라벨 프레임워크를 학습하기 위한 용도로 구현하였습니다.</p>
<p>google keep과 같은 레이아웃, 유사한 디자인으로 구성하였습니다.</p>
<p>UI 구성은 bootstrap 프레임워크로 구현하였습니다.</p>
<p>참고 사이트 : https://laravel.kr/docs/5.4/</p>
